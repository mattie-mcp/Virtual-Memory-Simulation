# Page Replacement System


## Requirements
* Node >= 6.11
* Grunt >= 1.0.1

## Building
This project consists of a client and server. Each must be built in order to use application.

* Clone repo
```console
git clone __link___
```

* Install dependencies from within client and server
```console
npm install
```

* Prep client
```console
cd client
grunt build
```

* Start server
```console
cd server
npm start
```


* Navigate to `localhost:3000`
